Fixed and Sticky Positions !!

  • Fixed Position

Applied in box 1 - An element with position: fixed; is positioned relative to the viewport, which means it always stays in the same place even if the page is scrolled. The top, right, bottom, and left properties are used to position the element.

  • Sticky Position !!

In CSS, position: sticky is a type of positioning that allows an element to "stick" to a specified position within its parent container when you scroll past it. The element remains in its normal flow until it reaches the specified scroll position, after which it behaves like a fixed element, sticking to that position until the parent container is out of view.

  • transform , filter or prespective properties can also make an elemet appear as positioned
